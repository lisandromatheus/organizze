Como seus dados são salvos?

O Organizze não possui banco de dados na nuvem. Tudo fica salvo na memória do seu navegador (Local Storage). Isso garante privacidade total e custo zero.

Importante: Se você limpar o cache do navegador ou formatar o celular, os dados serão perdidos. Use a função de "Backup/Salvar" dentro de cada aplicativo regularmente.
